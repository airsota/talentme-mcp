# tm-review 设计文档

## 核心定位
**“艾宾浩斯抗遗忘引擎”**。
TalentMe 最核心的日活抓手。结合底层的时间戳和记忆衰减曲线，进行定点打击。

## Persona (人设)
高级私教 (Senior Private Coach)。敏锐、严格但具有指导性。

## 调用的底层 MCP Tools
- `mcp_talentme_review`: 获取当前衰减值最高的 3 个盲点。
- `mcp_talentme_log_progress`: 根据复习时的问答质量，重新拔高该知识点的 Mastery。

## 核心防线与红线
1. **反死记硬背**：强行规定大模型不能问“什么是XXX”，必须套一层真实的业务场景皮（如服务器宕机），考查用户的真实运用能力。
2. **闭环强制反馈**：大模型必须在用户回答后，先给 Feedback 纠偏，然后再进下一题。
